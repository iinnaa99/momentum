## 💡 주요 기능

| 기능           | 내용                                                    |
| -------------- | ------------------------------------------------------- |
| 로그인         | Local storage를 통한 로그인 및 로그인 정보 저장         |
| 할 일          | Local storage를 통한 할 일 리스트 저장 및 삭제          |
| 현재 위치 날씨 | 현재 위치를 기반으로 날씨 API를 사용하여 날씨 정보 표시 |
| 시계           | 현재 시간 표시                                          |
| 명언           | 새로 고침 시 새로운 명언 표시                           |
| 배경화면       | 새로 고침 시 새로운 배경화면 표시                       |
